    Bugsense for WPF

    This projects makes it easy to learn about crashes "out there". With literally one line of code you can hook up Bugsense.WPF your application. When it crashes it will send a crash report Bugsense where you can use their easy-to-use but yet power ful analysis software to: 1. See what crashes are most common 2. Get in-dept knowledge about the crash in form a stacktrace among a lot of other information.

    Head over to http://www.bugsense.com/ for more information!

    Getting Bugsense.WPF

    Bugsense.WPF is available in the NuGet feed. See the package here: http://nuget.org/packages/Bugsense.WPF

    You will need to register for free to get an API-key at http://www.bugsense.com/
